# Decision Systems Portfolio

This is a recruiter-facing dashboard for two degree-apprenticeship evidence projects:

- **MARS**: Market Adaptive Risk System
- **Meridian**: Adaptive Capital Allocation Engine

The portfolio theme is:

> I build explainable decision engines for complex systems.

## Run locally

From this folder:

```bash
python -m http.server 8080
```

Then open:

```text
http://localhost:8080
```

## Deploy later

This is a static site, so it can be deployed to GitHub Pages, Netlify, Vercel, or any simple static host.
